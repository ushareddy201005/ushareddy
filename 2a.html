Understanding HTML Tables

In HTML, tables are created using the <table> tag. The table is structured with rows (<tr>) and cells, which can be either header cells (<th>) or data cells (<td>). Attributes like border, rowspan, and colspan can be used to enhance the table's appearance and structure.

Basic Table

Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3

Table with Rowspan

The rowspan attribute allows a cell to span multiple rows.

Header 1 Header 2 Header 3
Rowspan 2 Rows Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 2 Row 2, Cell 3

Table with Colspan

The colspan attribute allows a cell to span multiple columns.

Header 1 Header 2 Header 3
Row 1, Cell 1 Colspan 2 Columns
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3

Complex Table Example

This table combines both rowspan and colspan attributes.

Rowspan 2 Rows Row 1, Cell 2 Colspan 2 Columns
